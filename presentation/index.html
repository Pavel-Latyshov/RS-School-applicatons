Programming languages
Definition: programming language is a formal language comprising a set of strings that produce various kinds of machine code output. Programming languages are one kind of computer language, and are used in computer programming to implement algorithms.
Traits often considered important for what constitutes a programming language include:
  • Function and target
  • Abstractions
  • Expressive power
